# 🥃 Kadeh

Selam! Kadeh'e hoş geldin. 👋

**Kadeh**, alkol kültürüne yeni başlayanlar, merak edenler ve "Bunu sek mi içmeli yoksa karıştırayım mı?" diye düşünen herkes için hazırladığımız **kapsamlı bir Türkçe alkol rehberidir**. 

Burada arkada çalışan karmaşık veritabanları veya sunucular yok. Sadece saf içerik, güzel bir tasarım ve ışık hızında bir deneyim var! 🚀

## 🌟 Neler Var?

Kadeh'te aradığın (veya yeni keşfedeceğin) herhangi bir içkinin:
- **Tarihçesini ve Hikayesini** (Nasıl ortaya çıkmış? Kültürdeki yeri ne?)
- **Üretim sürecini** (Neden yapılıyor? Nasıl dinlendiriliyor?)
- **Tadım profilini** (Kokusu nasıl? Damakta nasıl bir his bırakıyor?)
- **Nasıl içilmesi gerektiğini** (Raconu nedir? Sek mi, kokteyl mi? Hangi kadehte?)
- **Kafa dozunu ve etkisini** (Ne zaman çarpar, ertesi gün üzer mi?)
- Hatta **hangi yemekle**, **hangi müzikle** ve **hangi mevsimde** iyi gideceğini bulabilirsin.

## 🛠️ Nasıl Çalışıyor? (Tech Stack)

Bu proje olabildiğince basit ve hızlı kalması için **tamamen statik** olarak tasarlandı:
- **Framework:** React (Vite) ⚡
- **Stil:** Tailwind CSS 🎨 (Premium ve koyu bir "dergi" teması için)
- **Veri:** Statik JS/JSON dosyaları (`src/data/` klasörünün içinde yaşıyorlar)
- **Deploy:** Vercel veya Netlify üzerinden otomatik
- **Lisans:** MIT

## 🚀 Bilgisayarında Çalıştır

Projeyi kendi bilgisayarında ayağa kaldırmak çocuk oyuncağı:

1. Repoyu klonla:
   ```bash
   git clone https://github.com/GalipEfeOncu/kadeh.git
   ```
2. Klasöre gir:
   ```bash
   cd kadeh
   ```
3. Paketleri yükle:
   ```bash
   npm install
   ```
4. Geliştirme sunucusunu başlat:
   ```bash
   npm run dev
   ```
   Ve ta-da! `localhost:5173` adresinde proje karşında. 🎉
