## 📖 Projenin Felsefesi

Kadeh; alkol kültürüne yeni başlayanlar, merak edenler ve *"Bunu sek mi içmeli yoksa karıştırmalı mıyım?"* diye düşünen herkes için tasarlanmış **kapsamlı ve jargonsuz bir Türkçe alkol rehberidir**.

Bizce içmek, yalnızca alkol tüketmekten ibaret değildir. Her içki, doğduğu coğrafyanın, iklimin ve tarihin damıtılmış bir özetidir. Rakı'da Akdeniz sofrasının neşeli tınıları, viskide İskoç sisinin ve turba kokusunun sabrı, şarapta ise Anadolu'nun bin yıllık bağlarının emeği saklıdır.

Kadeh, bu zengin mirası; ideal kadeh seçimi, servis ritüelleri, lezzet uyumları ve gastronomi eşleşmeleriyle birleştirerek, sommelier jargonundan uzak, samimi ve **premium bir dergi (editorial)** estetiğiyle kullanıcısına sunar.

## ✨ Öne Çıkan Özellikler

Kadeh, 11 ana kategori altında toplanmış 120'den fazla içki ve kokteyli barındırır. Her bir içki sayfasında şu detaylı başlıkları bulabilirsiniz:

- **📜 Lore & Tarihsel Hikaye:** İçkinin köken hikayesi, coğrafi çıkış noktası ve popüler kültürdeki yeri.
- **🍇 Üretim & Profil:** Üretimde kullanılan hammaddeler, damıtım/fermantasyon süreçleri, dinlendirme detayları ve duyusal tadım profili (koku, tat, bitiş).
- **🍷 Nasıl İçilir (Racon):** İdeal kadeh tipi, servis sıcaklığı, sek içim detayları ve en uyumlu 5 pratik kokteyl/karışım tarifi.
- **🍲 Gastronomi & Atmosfer Eşleşmeleri:** Uyumlu yemekler, yakıştığı mevsim ve o içkiye eşlik edecek müzik/his ortamı.
- **🧪 Doz & Hangover Analizi:** Alkol oranı (ABV), standart porsiyon miktarı, etki hızı ve ertesi gün hangover (akşamdan kalmalık) riski (1-5 arası puanlama).
- **🛒 Satın Alma & Keşif Rehberi:** Türkiye piyasasındaki bulunabilirliği, fiyat segmenti ve yeni başlayanlar/orta seviye/premium için somut marka önerileri.

## 🛠️ Teknoloji Yığını & Mimari

Kadeh, sunucu maliyeti olmadan ışık hızında çalışması ve tamamen taşınabilir olması için **tamamen statik** bir mimariyle tasarlanmıştır:

- **React 18 & Vite:** Bileşen tabanlı geliştirme ve ultra hızlı istemci tarafı (client-side) çalışma performansı.
- **Tailwind CSS:** Koyu (dark mode) ağırlıklı, premium dergi hissi veren, özel animasyon ve geçişlerle süslenmiş arayüz tasarımı.
- **React Router DOM v6:** Sayfalar arası akıcı geçişler ve lazy loading (Suspense) ile optimize edilmiş rota yönetimi.
- **Lucide React:** Minimalist ve estetik modern ikon kütüphanesi.
- **Zero-Backend (Statik Veri Yapısı):** Tüm veriler `src/data/` klasöründeki statik JS modüllerinde saklanır. Herhangi bir veritabanı sorgusu veya API gecikmesi olmaksızın, veriler anında istemci tarafında yüklenir ve aranabilir.
- **SEO & Arama Motoru Optimizasyonu:** Dinamik `document.title` güncellemeleri, optimize edilmiş `index.html` meta etiketleri, `sitemap.xml` ve `robots.txt` ile yüksek arama motoru uyumluluğu.

## 🚀 Yerel Kurulum ve Çalıştırma

Projeyi kendi bilgisayarınızda ayağa kaldırmak oldukça basittir:

1. **Repoyu klonlayın:**
   ```bash
   git clone https://github.com/GalipEfeOncu/kadeh.git
   ```

2. **Proje dizinine gidin:**
   ```bash
   cd kadeh
   ```

3. **Bağımlılıkları yükleyin:**
   ```bash
   npm install
   ```

4. **Geliştirme sunucusunu başlatın:**
   ```bash
   npm run dev
   ```
   Tarayıcınızda [http://localhost:5173](http://localhost:5173) adresine giderek projeyi canlı olarak görüntüleyebilirsiniz.

5. **Yayına hazır paket oluşturun (Build):**
   ```bash
   npm run build
   ```

## 🤝 Katkıda Bulunma

Kadeh'i birlikte büyütmekten mutluluk duyarız! En sevdiğiniz kokteyl tarifini eklemek, eksik bir alt türü tamamlamak veya içeriklerdeki hataları düzeltmek için katkı sağlayabilirsiniz.

> [!IMPORTANT]
> Projeye kod veya içerik katkısı yapmadan önce lütfen **[PROJECT.md](./PROJECT.md)** dosyasını inceleyin. Bu belgede veri şemaları, kodlama standartları ve tasarım kuralları detaylıca açıklanmıştır.

### Katkı Adımları:
1. Bu repoyu forklayın.
2. Yeni bir özellik dalı (feature branch) oluşturun: `git checkout -b feature/yeni-icki`
3. Değişikliklerinizi yapın. Veri ekliyorsanız `src/data/` altındaki ilgili kategori dosyasına şemaya uygun şekilde ekleyin.
4. Kodunuzun sorunsuz derlendiğinden emin olmak için `npm run build` komutunu çalıştırın.
5. Değişikliklerinizi commit edin: `git commit -m 'Ekle: Yeni Rakı Markası X'`
6. Dalınızı push edin: `git push origin feature/yeni-icki`
7. Bir Pull Request (PR) açın.
