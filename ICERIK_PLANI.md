## Veri Şablonu

Her içki bu alanlara sahip olmalı:

```js
{
  // Kimlik
  id, name, emoji, color, aliases, origin,

  // Sınıflandırma (filtre)
  ana_tur, alt_tur, gorunum, olgunlasma, aroma[], alkol_seviye,

  // Hikaye
  lore_short,               // 2-3 cümle
  lore_full: [              // 5 bölüm: Köken, Yayılma, Kültürel Etki, Modern Dönem, TR ile İlişkisi
    { baslik, metin }
  ],

  // Üretim
  hammadde, uretim_sureci, dinlendirme,

  // Tadım profili
  renk, koku, tat, finish, vucut,

  // Nasıl içilir
  abv_range,
  sek_detay: { kadeh, sicaklik, miktar, hazirlik },
  mix_detay: [{ isim, malzemeler[], hazirlanis, neden, ortam }],

  // Eşleşmeler
  yiyecek, muzik_his, mevsim,

  // Doz & Kültür
  kafa_dozu: { hafif, ideal, limit_ustu, ertesi_gun },
  sozler, yanlis_bilinenler, pop_kultur,
}
```
